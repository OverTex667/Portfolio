1. Inventory System :

Customizable inventory system with a hotbar, an inventory, and a trash system. The hotbar has 4 slot groups (from left to right: Card, Weapons, Utilities, Secondary) with 2 types of slots: - Visible empty slots (Card, Weapon, Secondary) - Invisible empty slots (Utility)

The inventory can hold up to 15 slots before it's full. It is sorted by slot type (Card first, Weapon second, Utility third, and Secondary last) and in alphabetical order. You can move a slot from the inventory to the hotbar only if they are the same type.

The trash system allows you to delete a slot only when inside the designated area (here, the red block).

2. NPC area :

A system where NPCs walk around randomly., Pink blocks are spawners (where NPCs appear), Red blocks are corners. When an NPC reaches a red block, it changes direction by choosing a neighboring red block (so it stays on the path).

The number of NPCs on the map can be customized with a single variable:
local max_npc = 10 -- Maximum number of NPCs at the same time.
